A Menerend gomb megnyomása után megjelenik a fentebb kiválasztott város felé a mai napon leghamarabb még közlekedő (maximum) három vonat menetrendje.

Készletek

A kiválasztott segélyponton az alábbi készletekre van szükség. Minél kevesebb van valamiből annál nagyobb szükség van rá.

Új készlet felvitelénél a telítettség alapbeállítása 50%

Segítőknek/Önkénteseknek



Az API kulcs egy minimum 10 karakter hosszú, helyszínhez kötött 'jelszó', amellyel a fent kiválasztott segélypont készleteit lehet adminisztrálni.

Az API kulcs beírásával automatikusan megjelennek a készletek nyilvántartását segítő kezelőgombok és beviteli mezők.
