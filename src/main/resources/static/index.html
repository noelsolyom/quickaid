Készletek

A kiválasztott segélyponton az alábbi készletekre van szükség. A sávok az adott dolog aktuális telítettségét jelölik. Minél kevesebb van valamiből annál nagyobb szükség van rá.

Új készlet felvitelénél a telítettség alapbeállítása 50%

Az API kulcs egy minimum 10 karakter hosszú, helyszínhez kötött 'jelszó', mellyel az adott segélypont készleteit lehet adminisztrálni.

Az API kulcs beírásával automatikusan megjelennek a segéd-interfészek.
